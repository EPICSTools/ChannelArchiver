  <h3>Command Explanation:</h3>
  <ul>
    <li><B>List</B>: List all channels that match <I>pattern</I></li>
    <li><B>Info</B>: Show info on channels that match <I>pattern</I>
         or are in <I>names</I> list</li>
    <li><B>Get</B>: Get values for given channels that are within
        time range</li>
    <ul>
        <li><B>Plot</B>:
            Get simple online plot.
        <li><B>Spreadsheet</B>:
            Get a spreadsheet-type text file which
            you can save to your disk for further investigation.
        <li><B>Matlab-Spreadsheet</B>: 
            Same, but with slightly different formatting for Matlab.
            The result requires some typing in Matlab, check the
            comments of the generated file.
        <li><B>Matlab</B>:
            Get a Matlab command file. Easier to import into Matlab
	    but usually requires more memory.
        <li><B>Status</B>:
            Export the channel status (disconnected, ...).
            Not supported for all formats.
    </ul>
  </ul>
