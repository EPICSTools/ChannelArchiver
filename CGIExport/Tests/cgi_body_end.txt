  <h3>Command Explanation:</h3>
  <ul>
    <li><B>List</B>: List all channels that match <I>pattern</I></li>
    <li><B>Info</B>: Show info on channels that match <I>pattern</I>
         or are in <I>names</I> list</li>
    <li><B>Get</B>: Get values for given channels that are within
        time range</li>
    <ul>
        <li><B>Plot</B>: Get simple online plot.
        <li><B>Spreadsheet</B>: Get a spreadsheet-type text file which
                                you can save to your disk for further
                                investigation.
        <li><B>Excel</B>: Same, but with HTTP hints that ask the web
                          browser to directly open the file in Excel,
                          skipping the save-to-disk. Might not work
                          for your PC.
        <li><B>Matlab</B>: Get a Matlab command file.
        <li><B>Status</B>: Export the channel status
                           (disconnected, ...).
                           Not supported for all formats.
    </ul>
  </ul>
