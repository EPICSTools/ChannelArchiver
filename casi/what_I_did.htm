What I did

This is what I installed on a local machine to get things going with the scripting support.

For Linux it is much easier: Simply install the runtime and developement RPMs, no compilation required.

Your milage may vary.

TCL

tcl8.2.3/unix: 
  ./configure --enable-gcc --enable-threads --prefix /home/john2/kasemir
  gmake
  gmake install

TK

tk8.2.3/unix: 
  ./configure --enable-gcc --enable-shared --prefix /home/john2/kasemir
  gmake
  gmake install

BLT

blt2.4n: 
  ./configure --with-cc=gcc --prefix /home/john2/kasemir
  # Hack Makefile: OPT=-g -O2 -D_REENTRANT
  gmake 
  gmake install

Python

Python-1.5.2: 
  ./configure --with-thread --with-gcc --prefix /home/john2/kasemir

_After_ a first make, check Modules/Setup. Enable Tkinter, make tcl/tk/blt point to /home/john2/kasemir/[include|lib] 

  gmake clean
  gmake
  gmake install

Pmw

Unpack Pmw.0.8.4.tgz in /home/john2/kasemir/lib/python1.5

SWIG

  setenv CXX g++          # Set CXX to your C++ compiler
  ./configure --prefix /home/john2/kasemir --with-lang=PYTHON\
     --with-doc=HTML --with-tcl=/home/john2/kasemir\
      --with-py=/home/john2/kasemir
  gmake
  gmake install
