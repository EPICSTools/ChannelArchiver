Web Page Generation

Another possible application is the interactive or automated generation of dayly reports.
Images of those plots can then be put on Web pages etc. The directory ChannelArchiver/casi/tcl/ExampleFromLEDA shows a combination of tcl and perl tools:
  • A tcl tool beam.tk is used to create two statistical plots, summarizing the beam current, on-time, total charge per day. It can be run interactively, as well as with command line options.
  • Perl scripts are used to call the tcl tool and create web pages for the resulting images as well as an index page.
  • A shell script is used to call the perl tools once a day.
