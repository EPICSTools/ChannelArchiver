Archive Engine

This ArchiveEngine collects samples via ChannelAccess and stores them. This engine is intended to replace "chan_arch", the solaris-only prototype implementation. It reads the same configuration files, it should also generate binary-compatible data files.
The ArchiveEngine is portable, running on Win32 and various Unixes including Linux. Status and configuration are accessible via a built-in HTTP server, so it's not limited to single status GUI per engine, portable and networked.
