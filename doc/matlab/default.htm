Matlab Support

Matlab provides almost infinite possibilities for plotting and analyzing data. Many of the ChannelArchiver data retrieval tools can export data in a format suitable for Matlab (version 4.2 or later):
  • ArchiveExport, WinBrowser can create Matlab files
  • CGIExport allows users to download Matlab files
  • LibIO provides a MatlabExporter class in case you want to build C++ programs that include this function
  • casi includes a ml_dump.pl perl script that generates the same output format.
