WinBrowser

This tool is a WIN32 Browser for ChannelArchiver data files.
It provides good performance and features like interactive scolling/zooming. But unlike the CGI Tool or the scripting interface CASI it's not portable, so its support is now lower on the list.

Usage

When started with the "-a <archive name>" option, WinBrowser will load the channel list for that archive. Otherwise you select an archive with the button next to the Archive name.
Now you can select channels from the list and send them to the plot window by pressing the button. Initially the date/timerange specified in the "From:.. To:.." fields will be loaded. When panning or zooming the plot, more samples will be loaded from the archive as necessary.
Finally you can print a hardcopy of the plot window or export the data into files (currently in Speadsheet or GNUPlot format)
