This TCL extension is not (yet) build as part of the standard makesystem.

Instead, there is a MS Visual C++ project file and a Makefile for Unix:

  • Adjust the Makefile (or project) to point to the correct path names for EPICS base, extensions and TCL
    (The project file assumes that you installed tcl/tk in c:\Progra~1\tcl. Check that you installed the full developement version of tcl/tk, including include and library files!)
  • Try to build the extension.
  • Configure atac.tcl to load the extension.

If you should succeed with building atac, continue here: ../doc/atac/default.htm
