The main.cfg file holds a test configuration for archiving the channels of the excas server.

The example CA server, excas, is part of EPICS base.
Sources are under base/src/cas/example/simple, the binary is usually found in base/bin/$HOST_ARCH.

To test this:

  1. Run excas
  2. Run the Engine as e.g.
    ../O.Linux/ArchiveEngine main.cfg dir
  3. Watch the engine with a web browser pointed to http://localhost:4812
  4. After some time, stop the engine via http://localhost:4812/stop
  5. Use the ArchiveManager to look at the samples, e.g.
    ../../Manager/O.Linux/ArchiveManager dir
    ../../Manager/O.Linux/ArchiveManager dir -c fred
    ...
